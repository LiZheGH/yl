注意: 设置公式的科目名称的计算方法中,只能够采用

+ - * / ( ) sum avg

上述符号.sum(科目名称)表示汇总[全院]之外的某些科室的某个科目的汇总值,并存入[全院]的记录中.

同理 avg则是[全院]之外的某些科室的某个科目的平均值,并存入[全院]的记录中.

例如: 出院病人好转治愈率 的公式: (出院病人好转数+出院病人治愈数)*100/出院人数总计

如果计算率,必须*100否则以小数显示.

再如: 员工满意度 的公式: avg(员工满意度)

说明:汇总公式不能与其他预算符号混合使用!
